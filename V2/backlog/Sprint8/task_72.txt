Описание задачи

В этих лекциях и в интернете найти ответы на вопросы:

1. Что такое реляционная база данных? - база данных имеющая табличное представление данных. Под конкретные данные
имеется таблица, под конкретные данные имеется строка и столб.

2. Что такое СУБД? - это ПО, для создания, управления, хранения и обработки данных в БД.
Основная задача - удобный доступ к данным

3. Что такое SQL? - Cтруктурированный язык запросов. Он используется для работы с реляционными БД.
Используется для создания, удаления, анализа, изменения, извлечения данных. А также управления структурой данных

Категории команд SQL:

DDL (Data Definition Language) — команды для определения структуры (CREATE, ALTER, DROP).
DML (Data Manipulation Language) — команды для работы с данными (SELECT, INSERT, UPDATE, DELETE).
DCL (Data Control Language) — команды для управления доступом (GRANT, REVOKE).
TCL (Transaction Control Language) — команды для управления транзакциями (COMMIT, ROLLBACK).

11. Что такое первичный ключ (primary key)?
Набор полей и одно поле, которое характеризует уникальность данной строки. По
первичному ключу будет просто найти необходимого пользователя

12. Что такое вторичный ключ (foreign key)?
Это поле, которое в другой таблице отображается как первичный ключ, нужен для того
чтобы получать данные с другой таблицы

13. Примеры команды SQL join.
Бывает 1:1 связь условно - одна машина - один владелец машины
Бывает 1:M связь условно - один дом - много собственников
Бывает М:М словно условно - много студентов - много книг в библиотеке (каждый может взять любую книгу, если она есть)
