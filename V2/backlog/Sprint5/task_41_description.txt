**Контекст задачи (краткое описание) **
Поменять формат даты в ответе на "yyyy-MM-dd".

**Что было сделано и как? (краткое описание)**
Вспомнил, что аннотация @RequiredArgsConstructor(access = AccessLevel.PACKAGE) - создает конструктор только с final
полями и конструктор будет с модификатором доступа дефолтный

Применил аннотацию  @JsonFormat(shape = JsonFormat.Shape.STRING, pattern = "yyyy-MM-dd")
библиотеки jackson, чтобы отформатировать поля для вывода JSON

**Зачем это было сделано? Чему научился в процессе выполнения задания?**

@JsonFormat — это аннотация Джексона, которую мы используем, чтобы указать,
как форматировать поля и/или свойства для вывода JSON.

@JsonFormat(shape = JsonFormat.Shape.STRING, pattern = "yyyy-MM-dd") для форматирования LocaleDateTime
